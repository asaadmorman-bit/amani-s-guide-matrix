# Security Policy

## Supported Versions

We actively enforce zero-trust cryptographic security models on the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

Do not open public GitHub issues for security anomalies. Please route encrypted cryptographic reports regarding pipeline vulnerabilities or license bypasses straight to: `security@amaniguide.internal`
